# Image Placeholders

Please add the following images to this directory:

1. **hero-bg.jpg** - Background image for the home page (canyon/nature scene)
2. **profile.jpg** - Profile photo (circular, professional headshot)
3. **research-behavior.jpg** - Research project thumbnail
4. **photo-1.jpg** to **photo-6.jpg** - Photography gallery images

You can use any images you like. For now, the website will show broken image placeholders until you add these files.

## Recommended Image Sizes:
- hero-bg.jpg: 1920x1080px or larger
- profile.jpg: 400x400px (square)
- research-behavior.jpg: 800x600px
- photo-*.jpg: 800x600px each
